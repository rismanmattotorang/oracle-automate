Payroll Run Execution in Oracle HCM

Run payroll with the Calculate Payroll flow (or a country-specific flow). The process reads element entries and the worker's salary, recurring, and one-time element values for the payroll relationship.

Payroll results are stored as run results per element; use View Person Process Results to inspect a single worker's element results for a payroll period.

Transfer results to the General Ledger via Transfer to Subledger Accounting and Create Accounting; payroll costing maps element eligibility to accounts so the entries post to GL_JE_LINES.
